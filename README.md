# py-candlestick-screener

## Installation.

```bash
pip install virtualenv
```

```bash
virtualenv venv
```
Activate venv
```bash
source venv/bin/activate
```
Install dependencies
```bash
pip install -r requirements.txt
```
Run 
```bash
flask run
```
